Unit Converter

How to Use Unit Converter:

1. Enter the value you want to convert in the "Enter value" box

2. Select the unit you are converting from in the first dropdown menu

3. Select the unit you want to convert to in the second dropdown menu

4. Click the "Convert Unit" button

5. The converted value will appear below the button

Example: Value: 2, From Unit: Kilograms, To Unit: Grams

Result: 2 kg = 2,000.00 g
